Personal note

This is a pretty basic crud using the MEAN stack with Angular 4.0.0. I've been actively teaching myself for not so long ago. It's quite simple so I hope it can eventually help anyone who's in the learning phase as well.

Prerequisites

- First of all you will need angular-cli which requires both Node 6.9.0 or higher and NPM 3 or higher.

- Install MongoDB and create these two folders C:\data\db

- It will also be required: ExpressJS, body-parser, EJS, MongoJS, Express-History-API-Fallback and Materialize CSS. They can all be easily installed by running the npm install command.

Building

- ng build in order to create the dist folder.

Developing

- Run these guys in order to initialize MongoDB

"C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe"

"C:\Program Files\MongoDB\Server\3.4\bin\mongo.exe"

- node server and run on http://localhost:3000
